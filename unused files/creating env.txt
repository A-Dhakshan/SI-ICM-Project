Follow this procedure for windows

After installing python, Go to -> "start" and type "Manage App Execution Aliases". Go to it and turn off (both) "Python". Then in terminal

> py -m venv env_name
> .\env_name\Scripts\activate


Procedure to uninstall all packages in an environment. This leave only pip

pip freeze > requirements.txt

Now to remove one by one
pip uninstall -r requirements.txt

If we want to remove all at once then
pip uninstall -r requirements.txt -y
